# bohaoxing.com

Personal academic website of **Bohao Xing**, Doctoral Researcher at LUT University, Finland.

🔗 **Live site:** [bohaoxing.com](https://bohaoxing.com)

## About

This site showcases my research in affective computing, multimodal large language models, and privacy-preserving emotion understanding. It includes my publications, CV, teaching activities, and news.

## Built With

- [Jekyll](https://jekyllrb.com/) — static site generator
- [al-folio](https://github.com/alshedivat/al-folio) — base theme for academics
- Custom design: warm paper palette, Newsreader + Inter typography, editorial layout
- Hosted on [GitHub Pages](https://pages.github.com/)
